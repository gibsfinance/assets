# Attribution Provider Logos

This directory contains logo images for attribution providers displayed in the Attribution component.

## Required Logos

The following logo images need to be added to this directory:

- `dexscreener.png` - Dexscreener logo
- `coingecko.png` - Coingecko logo
- `uniswap.png` - Uniswap logo
- `pulsechain.png` - PulseChain logo
- `pulsex.png` - PulseX logo
- `balancer.png` - Balancer logo
- `internet-money.png` - Internet Money logo
- `pumptires.png` - PumpTires logo
- `trustwallet.png` - Trust Wallet logo
- `tokensex.png` - TokenSex logo
- `uma.png` - UMA logo
- `optimism.png` - Optimism logo
- `pls369.png` - PLS369 logo

## Guidelines

- Images should be PNG format with transparent backgrounds when possible
- Recommended size: 100x100px or larger (they will be scaled down)
- Images should be square or horizontally oriented
- File names should match exactly as listed above

## Missing Images

Images that are missing will not display (hidden via CSS `onerror` handler), so you can add them incrementally.

## Sources

You can find logos from:
- Official websites of each provider
- Trust Wallet assets repository (available in submodules/trustwallet)
- Smoldapp token assets repository (available in submodules/smoldapp-tokenassets)
- PulseChain assets repository (available in submodules/pulsechain-assets)

Example: Optimism logo is available at `submodules/trustwallet/blockchains/optimism/info/logo.png`
